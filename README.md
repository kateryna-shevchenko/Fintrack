# FinTrack 

FinTrack is a personal finance management application that enables users to track expenses, monitor spending habits, and gain insights into their financial activity. The project is containerized with Docker, providing a consistent and easy-to-deploy development environment.

## Features

-  Financial dashboard
-  Expense and income tracking
-  Data visualization and analytics
-  Transaction filtering
-  Responsive user interface
-  Dockerized setup for easy deployment

## Tech Stack

- HTML
- CSS
- JavaScript
- React
- Node.js
- Express.js
- MySQL
- Docker

## Getting Started

### Using Docker

Clone the repository:

```bash
git clone https://github.com/Dares1868/Fintrack.git
cd Fintrack
```

Build and start the containers:

```bash
docker compose up --build
```

The application will be available at:

```text
http://localhost:3000
```

### Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
